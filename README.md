# Atom Devquote

Show a random developer quote within Atom.

![Atom Devquote](http://i.imgur.com/P8lAyWZ.gif)


## Usage

In the `Command Palette`, search for `Atom Devquote: Show Random Quote`, or press `Ctrl-Alt-Shift-D`.
